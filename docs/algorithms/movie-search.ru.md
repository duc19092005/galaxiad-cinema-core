# Алгоритм поиска фильмов

В этом документе объясняется, как работает поиск фильмов в системе.

## Базовый поиск

- Поиск фильмов по названию, описанию, режиссеру или актерам
- Использует SQL LIKE/полнотекстовый поиск для базовых запросов
- Результаты сортируются по релевантности

## Расширенный поиск

- Фильтр по жанру, формату, возрастному рейтингу, кинотеатру
- Фильтр по дате и времени сеанса
- Комбинация нескольких критериев поиска одновременно
