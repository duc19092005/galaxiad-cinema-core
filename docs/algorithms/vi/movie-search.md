# Thuật Toán Tìm Kiếm Phim

Tài liệu này giải thích cách tìm kiếm phim hoạt động trong hệ thống.

## Tìm Kiếm Cơ Bản

- Tìm kiếm phim theo tên, mô tả, đạo diễn hoặc diễn viên
- Sử dụng SQL LIKE/Full-Text Search cho các truy vấn cơ bản
- Kết quả được sắp xếp theo mức độ liên quan

## Tìm Kiếm Nâng Cao

- Lọc theo thể loại, định dạng, độ tuổi, rạp chiếu
- Lọc theo ngày và giờ chiếu
- Kết hợp nhiều tiêu chí tìm kiếm cùng lúc
