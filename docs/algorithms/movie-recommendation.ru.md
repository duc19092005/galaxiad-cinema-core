# Алгоритм рекомендаций фильмов

В этом документе объясняется алгоритм рекомендаций фильмов для клиентов на основе поведения.

## Обзор

Система рекомендаций фильмов работает на трех уровнях:
1. **Рекомендации на основе поведения** — использует Qdrant (векторная база данных) для поиска похожих фильмов на основе истории просмотров клиента
2. **Рекомендации на основе контента** — SQL-запросы на основе жанров, форматов и возрастных рейтингов
3. **Резервная оценка** — оценки по умолчанию, когда недостаточно данных

## Процесс

1. Когда клиент входит в систему, система загружает историю просмотров и любимые жанры
2. Система ищет похожие фильмы в Qdrant
3. Если Qdrant не дает достаточно результатов, используется SQL в качестве резерва
4. Итоговая оценка является комбинацией нескольких факторов
