## 1. Учетные записи и контроль доступа

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS01** | Конфигурация только для администратора | Категории объектов, списки отделов и системная конфигурация могут управляться только учетными записями с ролью **Admin**. |
| **BS02** | Возраст регистрации клиента | Для регистрации в качестве Клиента необходимо быть не моложе **16 лет**. |
| **BS03** | Возраст регистрации сотрудника | Для регистрации в качестве Сотрудника необходимо быть не моложе **18 лет**. |
| **BS04** | Способы регистрации | Регистрация возможна через **email/пароль** или через **аккаунт Google**. |
| **BS05** | Доступ на основе роли | Каждый пользователь имеет роль. Пользователи могут видеть и выполнять только действия, соответствующие правам их роли. |
| **BS06** | Блокировка учетной записи | **Admin** может заблокировать учетную запись пользователя. Причина блокировки должна быть записана. |
| **BS07** | Безопасность сессии | Сеансы входа используют защищенные токены (JWT). Сеансы автоматически истекают. |
| **BS08** | Безопасность при изменении прав | Если права пользователя изменяются во время активного сеанса, система выполняет принудительный выход из соображений безопасности. |

### Роли пользователей

| Роль | Бизнес-описание |
|:----:|:----------------|
| **Customer (Клиент)** | Просматривает фильмы, бронирует билеты, пишет отзывы, управляет своими бронированиями |
| **Cashier (Кассир)** | Продает билеты в кассе, обрабатывает платежи для посетителей |
| **Movie Manager (Менеджер фильмов)** | Добавляет, редактирует и удаляет фильмы в системе |
| **Theater Manager (Управляющий кинотеатром)** | Управляет кинотеатрами, залами, сменами сотрудников и расписанием сеансов |
| **Facilities Manager (Управляющий объектами)** | Управляет объектами кинотеатра, планировкой залов и оборудованием |
| **Admin** | Полный доступ: управляет пользователями, ролями, передачей прав, ваучерами и просматривает журналы аудита |

## 2. Кинотеатры и объекты

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS09** | Назначение управляющих кинотеатром | Каждому кинотеатру могут быть назначены **Управляющий кинотеатром** и **Управляющий объектами**. |
| **BS10** | Поддержка форматов зала | Зал может поддерживать один или несколько форматов фильмов (2D, 3D, IMAX, 4DX и т.д.). |
| **BS11** | Полная схема мест | Схема мест зала должна быть полной. Ряды и колонки должны быть непрерывными, без неожиданных пропусков. |
| **BS12** | Уникальное положение места | Каждое место в зале должно иметь уникальную позицию в сетке. |
| **BS13** | Уникальная метка места | Каждое место должно иметь уникальную метку. Два места не могут называться "A1". |
| **BS14** | Типы отделов | Кинотеатр может иметь отделы, такие как **Билетная касса** и **Касса закусок**. |
| **BS15** | Общие учетные записи отделов | Каждый отдел имеет общую учетную запись для входа кассиров. |

## 3. Управление фильмами

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS16** | Возрастной рейтинг фильмов | Каждый фильм должен иметь классификацию возрастного рейтинга. |
| **BS17** | Форматы фильмов | Фильм может быть доступен в нескольких форматах (2D, 3D, IMAX, 4DX и т.д.). |

| Метка | Значение |
|:----:|:---------|
| **P** | Подходит для всех возрастов |
| **K** | Детям до 13 лет требуется сопровождение родителей или опекунов |
| **T13** | Для зрителей от 13 лет и старше |
| **T16** | Для зрителей от 16 лет и старше |
| **T18** | Для зрителей от 18 лет и старше |
| **C** | Не разрешен к распространению |

## 4. Расписание сеансов

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS19** | Назначение расписания | Каждый сеанс принадлежит одному фильму, одному залу и одному формату фильма. |
| **BS20** | Совместимость форматов | Формат фильма сеанса должен поддерживаться выбранным залом. |
| **BS21** | Интервал уборки | Между окончанием одного сеанса и началом следующего в одном зале должно быть не менее **15 минут**. |
| **BS22** | Нет сеансов в прошлом | Нельзя назначить сеанс в прошлом. |
| **BS23** | Нет пересекающихся сеансов | Сеансы в одном зале не могут пересекаться друг с другом. |
| **BS24** | Часы работы | Все сеансы должны проводиться в часы работы кинотеатра: **с 6:00 до 2:00** следующего дня (местное время Вьетнама). |
| **BS25** | Часовой пояс Вьетнама | Все время расписания сеансов соответствует местному времени Вьетнама (UTC+7). Бэкенд хранит время в UTC и конвертирует для отображения. |

## 5. Бронирование билетов

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS26** | Минимальное количество мест | Клиент должен выбрать не менее **1 места** для бронирования. |
| **BS27** | Максимальное количество мест | Клиент не может выбрать более **10 мест** в одном заказе. |
| **BS28** | Нет повторяющихся мест | Одно и то же место не может быть выбрано дважды в одном заказе. |
| **BS29** | Недоступные места | Места, уже оплаченные или временно забронированные другим клиентом, не могут быть выбраны. |
| **BS30** | Избегание одиночных мест | Клиенты не должны выбирать места таким образом, чтобы оставалось только **одно пустое место** между занятыми местами в ряду. |
| **BS31** | Временное удержание мест | Когда клиент выбирает места и начинает оформление заказа, эти места временно удерживаются. Если оплата не завершена вовремя, места снова становятся доступными. |
| **BS32** | Способ оплаты | Онлайн-платежи обрабатываются через **VNPay**. Продажи в кассе могут обрабатываться наличными кассиром. |
| **BS33** | Ценообразование на сервере | Окончательная цена билета всегда рассчитывается на бэкенде. Система не доверяет цене, отправленной из браузера. |
| **BS34** | Снимок цены | Примененные цены и акции сохраняются в виде снимка в заказе для исторической справки. |
| **BS35** | Подтверждение бронирования | Бронирование действительно только после успешной оплаты. Клиент получает код бронирования и может скачать электронный билет в формате PDF. |
| **BS36** | Изменение бронирования | Клиенты могут изменить или отменить бронирование не позднее чем за **2 часа до сеанса**. Может взиматься плата за отмену. |
| **BS37** | Срок возврата средств | Возврат средств обрабатывается на исходный способ оплаты в течение **5–7 рабочих дней**. |

### Статусы заказов

| Статус | Бизнес-значение |
|:------:|:----------------|
| **Pending (В ожидании)** | Места выбраны, но оплата еще не завершена |
| **Booked (Забронировано)** | Оплата прошла успешно, билет действителен и подтвержден |
| **Canceled (Отменено)** | Заказ отменен клиентом или из-за истечения времени оплаты |
| **Refunded (Возвращено)** | Платеж возвращен (например, техническая проблема или отмена сеанса кинотеатром) |
| **Completed (Завершено)** | Клиент посетил сеанс, билет был проверен |

## 6. Цены и акции

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS38** | Базовая цена | Каждый формат фильма имеет базовую цену билета. Это отправная точка для всех расчетов цены. |
| **BS39** | Скидка по формату | Кинотеатр может установить процент скидки для определенных форматов фильмов. |
| **BS40** | Наценка по формату | Кинотеатр может установить процент наценки для определенных форматов и сегментов клиентов. |
| **BS41** | Типы акций | Акции могут быть: **Фиксированная цена билета**, **Процентная скидка**, **Фиксированная скидка** или **Наценка**. |
| **BS42** | Область действия акции | Правило акции может быть ограничено по формату, кинотеатру, залу, сегменту клиентов, диапазону дат, временному интервалу и дням недели. |
| **BS43** | Приоритет акции | Если несколько акций конфликтуют, применяется акция с **наивысшим приоритетом**. |
| **BS44** | Разрешение фиксированной цены | Если совпадают несколько правил фиксированной цены, используется только правило с наивысшим приоритетом. Если приоритеты равны, побеждает самая низкая цена. |
| **BS45** | Исключение праздников | Кампания акции может быть настроена на **неприменение в праздничные дни**. |
| **BS46** | Минимальная цена | Окончательная цена билета не может быть ниже нуля. |
| **BS47** | Публичные предложения | Активные акции отображаются публично на странице предложений и **применяются автоматически** при бронировании. Код купона не требуется. |

### Специальные цены для клиентов

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS48** | Цена для студентов | Студенты могут получить фиксированную специальную цену билета при наличии права. |
| **BS49** | Скидка для пожилых | Пожилые граждане получают скидку не менее **20%** на цены билетов. |
| **BS50** | Скидка для инвалидов | Лица с тяжелой инвалидностью получают скидку не менее **50%**. |
| **BS51** | Специальная помощь | Лица с особо тяжелой инвалидностью получают **бесплатные билеты**. |

> Специальные цены применяются при покупке в кассе при предъявлении действительного удостоверения личности.

## 7. Ваучеры и бонусные баллы

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS52** | Создание ваучера | Admin может создавать ваучеры с названием, описанием, суммой/процентом скидки, стоимостью в баллах, количеством и сроками действия. |
| **BS53** | Запас ваучеров | Каждый ваучер имеет общее количество и остаток на складе. |
| **BS54** | Погашение ваучера | Клиент может погасить ваучер, используя свои бонусные баллы. |
| **BS55** | Ваучер закончился | Ваучер не может быть погашен, если его остаток равен нулю. |
| **BS56** | Истечение срока ваучера | Ваучер не может быть погашен вне срока его действия. |
| **BS57** | Недостаточно баллов | Клиент не может погасить ваучер, если у него недостаточно бонусных баллов. |
| **BS58** | Списание баллов | Бонусные баллы списываются немедленно при погашении ваучера. |

## 8. Сотрудники и управление сменами

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS59** | Типы сотрудников | Сотрудники классифицируются как **Полная занятость** или **Частичная занятость**. |
| **BS60** | Продолжительность полной смены | Смена полной занятости должна длиться ровно **8 часов**. |
| **BS61** | Продолжительность неполной смены | Смена частичной занятости должна длиться ровно **4 часа**. |
| **BS62** | Гибкость ротационной смены | Ротационные смены не имеют фиксированной продолжительности — они могут быть любой длины. |
| **BS63** | Право на неполную смену | Сотрудники с частичной занятостью могут регистрироваться только на неполные смены (4ч) или ротационные смены не длиннее 4 часов. |
| **BS64** | Право на полную смену | Сотрудники с полной занятостью, регистрирующиеся на смену короче 8 часов, должны предоставить письменное объяснение. |
| **BS65** | Назначение кинотеатра | Сотрудники могут регистрироваться только на смены в кинотеатре, к которому они прикреплены. |
| **BS66** | Смена в часы работы | Все смены должны быть в часы работы кинотеатра (с 6:00 до 2:00). |
| **BS67** | Нет повторной регистрации | Сотрудник не может зарегистрироваться на одну и ту же смену дважды. |
| **BS68** | Утверждение смены | Регистрации смен начинаются со статуса **В ожидании** и требуют утверждения руководителем или Admin. |
| **BS69** | Обязательный вход | Сотрудник должен отметить начало своей смены. |
| **BS70** | Обязательный выход | Сотрудник должен отметить окончание своей смены. |
| **BS71** | Регистрация лица | Сотрудники могут зарегистрировать свое лицо для входа по распознаванию лица. |
| **BS72** | Расчет заработной платы | Зарплата рассчитывается на основе отработанных часов и почасовой ставки сотрудника. |

## 13. Юридические и нормативные требования

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS90** | Приоритет вьетнамских фильмов | Сеансы вьетнамских фильмов имеют приоритет с **18:00 до 22:00**. |
| **BS91** | Время сеансов для младше 13 лет | Сеансы для клиентов младше 13 лет должны заканчиваться **до 22:00**. |
| **BS92** | Время сеансов для младше 16 лет | Сеансы для клиентов младше 16 лет должны заканчиваться **до 23:00**. |
| **BS93** | Запрет записи | Клиенты не должны незаконно записывать фильмы в кинотеатре. |
| **BS94** | Запрет нарушений | Клиенты не должны создавать беспокойство или мешать другим клиентам. |
| **BS95** | Запрет курения | Табак и электронные сигареты не разрешены в кинотеатре. |
| **BS96** | Запрет оружия | Оружие, легковоспламеняющиеся материалы и токсичные вещества запрещены. |
| **BS97** | Запрет животных | Домашние животные не допускаются в кинотеатр (кроме служебных животных). |
| **BS98** | Запрет еды извне | Еда и напитки извне не допускаются без разрешения. |
| **BS99** | Обязательство конфиденциальности | Личные данные клиентов не продаются и не передаются третьим лицам без согласия. |
| **BS100** | Использование cookie | Веб-сайт использует cookie для улучшения пользовательского опыта. Клиенты могут управлять настройками cookie в своем браузере. |

| Код | Название правила | Содержание |
|:---:|:-----------------|:-----------|
| **BS101** | В зале должны быть места | Зал должен содержать как минимум одно место. |
| **BS102** | Непрерывные ряды | Ряды мест должны быть заполнены непрерывно. Ряд не может пропускать позицию 1 и переходить к позиции 3, если позиция 2 отсутствует. |
| **BS103** | Непрерывные колонки | Колонки мест должны быть заполнены непрерывно без пропусков в середине. |
| **BS104** | Нет пересекающихся позиций | Два места не могут иметь одинаковую позицию в сетке. |
| **BS105** | Нет повторяющихся меток | Два места не могут иметь одинаковую метку в одном зале. |
